# Wordbound 1.1: First-session and retention sprint

## Added

- First-run onboarding with explorer name and crest
- Clear explanation of missions, adaptive learning and daily review
- Five-word daily review card on the home screen
- Review selection prioritising due, weak, mistaken and unseen words
- First daily completion XP and coin bonus
- Separate review results screen
- Backup download and restore controls inside explorer settings
- Versioned offline cache for all new modules

## Why

The first vertical slice proved the game loop, but it did not yet explain itself or create a strong reason to return. This sprint improves comprehension on the first visit and establishes a short daily learning ritual.

## Preserved

- All five Whispering Woods missions
- Glitch Thorn boss
- Mission XP, coins, ranks and restoration
- Parent report
- Speech, hints, shields and focus runes
- Existing local progress

## Next recommended sprint

Build **Chapter Two: Clockwork City** only after mobile playtesting confirms that onboarding, review and the first five missions are understandable and enjoyable.
